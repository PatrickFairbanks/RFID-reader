Writes an array of 8-bit bytes (the LSBs of the array elements) to the register of the device with the specified addresses. No array bound checks are done. Returns 1 if the operation succeeds and is ACKed at all stages; 0 else. Function may only be called if the device is set as an I2C master; otherwise, undefined behaviour occurs.

Returns

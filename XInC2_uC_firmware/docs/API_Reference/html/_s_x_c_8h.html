Detailed Description

Basis for defining XInC2 functionality, such as "under-the-hood" interface and std library functions.

Optimization macros for SXC library and XInC2 processor. asm definitions for XInC2 instructions that don't model well in LLVM. Definitions of debugging/assertion switches. Effectively, three target platforms: XInC2, TEST_HARNESS (internal debugging ), and all others. XInC2 simply uses the asm and hardware, TEST_HARNESS provides no implementations and simply closely emulates the XInC2 interface, and all others rely on POSIX.

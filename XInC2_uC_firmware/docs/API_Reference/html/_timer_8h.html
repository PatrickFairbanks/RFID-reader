Detailed Description

Interface for accessing TimerA and TimerB functions on the XInC2 processor. NOT thread-safe; timers do not have internal semaphores in hardware and the user MUST manually manage semaphores to protect their state.

Reads the specified timer capture/compare register value at the time of function call.

Parameters
[in]timerThe timer to target.

Returns
The CCR's value at the time of function call. This is the timer value latched on a capture event in capture mode, and the compare register value in compare mode. Returns 0 if something has gone awry, which unfortunately can also mean that the CCR value was just 0.

Detailed Description

Interface for accessing the GPIO ports on the XInC2 processor. These functions are NOT thread-safe: manual semaphore management is required. See the XInC2 manual for hardware layout for a much better understanding.

SXC_INLINE void gpio_set_config ( uint16_t  cfg,
enum GPIO_port_name  port 
)

Configures the selected GPIO port pins' directionality based on the provided configuration. CLOBBERS THE OUTPUT BUFFER REGISTER (LSB). Call gpio_set_config((gpio_get_config() & 0xFF) | cfg) to avoid this, if you care.

Parameters
[in]cfgDesired pin configuration. Lower byte clobbers output buffer register. Set bits in the upper byte cause that pin to be an output; cleared bits correlate to input pins. Because this is the MSB, bit8 of the entire word corresponds to pin0, bit9 to pin1, and so on.
[in]portThe selected GPIO port.

SXC_INLINE void ADC_set_channel_nums ( uint16_t const *  nums)
inline

Sets the 2-bit channel number(s) for the ADC to read from. Reads in 1 channel number if the ADC is in single conversion mode. If it is in continuous sampling mode, then as many channel numbers are read in as the ADC is configured for (3 or 4). Only the least significant 2 bits from each number are read; all others are ignored. No array boundary checking is done. Behaviour is unpredictable if the ADC is not configured before calling this function.

Parameters
[in]numsPointer to desired sampling channel numbers

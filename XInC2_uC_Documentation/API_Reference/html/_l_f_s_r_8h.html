Detailed Description

Interface for accessing the linear feedback shift registers on the XInC2 processor. These functions are NOT thread-safe: manual semaphore management is required. See the XInC2 manual for hardware layout for a much better understanding.

Configures the selected LFSR to behave according to the provided configuration. See XInC2 manual for details.

Parameters
[in]lfsr_msb4-bit integer representing the MSB of the LFSR. Also equal to desired LFSR length-1. Bits beyond bit3 are ignored.

[in]fb_modeThe LFSR feedback mode to use.

[in]lfsrThe LFSR to configure.
